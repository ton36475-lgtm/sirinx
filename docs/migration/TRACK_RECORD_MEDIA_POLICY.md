# Track Record Media Policy

## Purpose

Ensure SIRINX Track Record pages use premium, real-world, permissioned imagery without implying unverified performance, ROI, or package truth.

## Allowed Media

- Real solar carport photos.
- Smart hotel or commercial energy-management installation photos.
- BESS, EV charger, EMS, inverter, meter, cable tray, and mounting detail photos.
- Monitoring screenshots with redaction.
- Comparable commercial/industrial clean-tech installation imagery with proper labels.

## Blocked Media

- Gambling, casino, lottery, betting, or CHOKMA assets.
- AI-generated site images presented as real work.
- Stock images presented as SIRINX work.
- Unapproved customer logos, addresses, faces, meter serials, bills, or signage.
- Low-trust promo-collage styling.

## Required Metadata

- asset path
- source owner
- permission status
- redaction status
- claim status of nearby copy
- linked hardware/pilot/ROI/track-record record
- reviewer and approver

## Rollback Note

If an unapproved image appears on a public Track Record page, remove it, restore the last approved asset, downgrade nearby claims to `Review Required`, and preserve before/after evidence.

## Audit Trail Note

Save original asset, redacted asset, source/permission note, linked governance record, reviewer, approver, and rollback target.
