# Vercel Deploy Checklist For Design Lane

## Purpose

Provide a safe preview checklist for design-lane prototypes before they are considered for canonical runtime adoption.

## Checklist

- mobile-first layout reviewed
- hero text remains legible at narrow widths
- Track Record imagery remains real-world and permission-aware
- no locked fact mutation
- no guaranteed ROI claims
- no unrelated brand contamination
- motion remains optional and performance-safe
- preview build is review-only and does not change production

## Runtime Expectations

- optimize hero media before preview
- avoid oversized animation bundles
- preserve LCP and CLS targets
- treat preview URLs as review artifacts only

## Handoff Rule

Any approved preview outcome must still be translated back into canonical repo files and passed through Validator and Delivery before server handoff.
